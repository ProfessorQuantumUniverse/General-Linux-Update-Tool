# 🐧 Linux Update Script Generator

### Erstelle mit wenigen Klicks ein passendes Update-Skript für deinen Linux-Server.

Ein praktischer Web-Generator: wähle deine Distribution und Optionen aus und erhalte ein fertiges, sicheres Update-Skript zum Kopieren – ganz ohne selbst die Befehle nachzuschlagen.

## ✨ Features

- 🧩 Unterstützt gängige Paketmanager (apt, dnf, pacman …)
- 📋 Skript per Klick kopieren
- ⚙️ Optionen wie Autoremove & Cleanup
- 🌐 Läuft komplett im Browser, keine Installation nötig

## 🛠️ Tech-Stack

`HTML5` · `CSS3` · `JavaScript`
